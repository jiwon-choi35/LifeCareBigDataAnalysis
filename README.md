# 청년층 우울 위기 진단

질병관리청 **지역사회건강조사(CHS)** 2020부터 2024년, 즉 5개년 통합 데이터를 활용하여  
청년층(20~39세)의 우울증 발생에 영향을 미치는 요인을 다중 로지스틱 회귀 분석으로 규명한 프로젝트입니다.


---

## 연구 배경 및 목적

우울증은 최근 5년간 국내 정신질환 중 환자 수 1위(평균 980,888명)를 기록하며, 2020년 832,483명에서 2024년 1,106,603명으로 지속 증가하는 추세입니다. 특히 경제적 불안정과 수면 문제가 심각한 청년층을 대상으로 우울증 위험 요인을 데이터 기반으로 분석하고자 했습니다.

본 연구의 목적은 네 가지입니다.

1. 청년층 우울감 경험의 시계열적 추이 분석 및 유병 규모 진단
2. 경제활동 여부·수면시간과 우울 경험 간의 상관성 검증
3. 성별·만성질환 등 교란 변수를 통제한 상태에서의 **순수한 보호 효과** 측정
4. 분석 결과에 근거한 맞춤형 정책·프로그램 제안

## 분석 방법 및 절차

### 사용 환경

- **언어**: Python
- **라이브러리**: Pandas, NumPy, Statsmodels
- **원시 데이터 형식**: SAS (`.sas7bdat`)

### 분석 절차

**1. 자료 통합**  
5개년 SAS 파일을 `pd.concat()`으로 단일 DataFrame으로 병합 후 만 20~39세 연령 필터링

**2. 변수 전처리 및 이진화**

```python
df['depressed']        = df['mtb_01z1'].replace({2: 0}).astype(int)   # 우울감: 1=경험, 0=미경험
df['econ_active']      = df['soa_01z1'].replace({2: 0, 1: 1}).astype(int)
df['sleep_hours']      = df['mtc_10z1'].astype(float)                  # 연속형 유지
df['male']             = df['sex'].replace({1: 1, 2: 0}).astype(int)
df['has_diabetes']     = df['dbt_02z1'].replace({2: 0}).astype(int)
df['has_hypertension'] = df['hbp_02z1'].replace({2: 0}).astype(int)
```

**3. 결측치 처리**  
이진 값 범위를 벗어난 응답 및 수면시간 코드 77·99 제거

**4. 가중치 적용**  
개인 가중치(WGT)를 모든 통계 추정 및 모델 분석에 필수 적용  
→ 분석 결과가 특정 응답자가 아닌 **전국 청년 인구를 대표**함을 보장

**5. 기술 통계 및 카이제곱 검정**  
주요 변수 빈도·비율·평균·표준편차 산출 후, 독립변수-종속변수 간 단순 교차 연관성 사전 검증

**6. 다중 로지스틱 회귀 분석 (핵심 모델)**

```
# 모델 1 - 경제활동 여부
Logit(P(Depressed)) = β0 + β1(econ_active) + β2(male) + β3(has_diabetes) + β4(has_hypertension)

# 모델 2 - 수면시간
Logit(P(Depressed)) = β0 + β1(sleep_hours) + β2(male) + β3(has_diabetes) + β4(has_hypertension)
```

**7. 모델 진단**
- **Hosmer-Lemeshow 검정**: 모형 적합성 확인 (p > 0.05 → 적합)
- **VIF 진단**: 다중 공선성 확인 (전 변수 VIF ≤ 2.0 → 문제 없음)

## 분석 결과

### 모델 적합성

| 진단 항목 | 결과 | 기준 | 판정 |
|-----------|------|------|------|
| Hosmer-Lemeshow p-value | > 0.05 | > 0.05 | ✅ 적합 |
| 최대 VIF | ≤ 2.0 | < 10 | ✅ 다중공선성 없음 |
| 모든 변수 P-value | < 0.001 | < 0.05 | ✅ 통계적 유의 |

### 독립변수 결과

**경제활동 여부 → 우울감**

| 지표 | 값 |
|------|----|
| Odds Ratio (OR) | 0.8093 |
| 95% 신뢰구간 | [0.723, 0.775] |
| P-value | < 0.001 |
| 우울감 오즈 감소율 | **약 19.07%** |

성별·당뇨병·고혈압을 통제한 후에도, 경제활동 참여 청년은 비참여 청년 대비 우울감 경험 오즈가 약 **19.07% 감소**합니다. 역인과성 우려를 통제 변수로 해소하였으며, 경제활동 참여가 청년 정신건강에 유의미한 보호 효과를 가짐을 입증합니다.

**수면시간 → 우울감**

| 지표 | 값 |
|------|----|
| Odds Ratio (OR) | 0.7992 |
| 95% 신뢰구간 | [0.774, 0.823] |
| P-value | < 0.001 |
| 수면 1시간 증가 시 오즈 감소율 | **약 20.08%** |

통제 변수를 고려한 상태에서 수면시간이 1시간 늘어날수록 우울감 오즈가 약 **20.08% 감소**합니다. 수면 부족이 우울감 유발의 강력한 위험 요인임을 확인합니다.

### 통제변수 결과

| 변수 | OR | 해석 |
|------|----|------|
| 고혈압 | 1.7470 | 질환 있는 청년, 우울 위험 **74.7% 증가** |
| 당뇨병 | 1.6410 | 질환 있는 청년, 우울 위험 **64.1% 증가** |
| 성별(남성) | 0.5470 | 남성이 여성 대비 우울 오즈 **45.3% 낮음** |

## 결론

1. **구조적 우울 위험의 만성화**: 2020년 최고점 이후 소폭 하락했으나 2024년에도 약 12%의 높은 우울감 경험률을 유지하고 있어, 청년층 우울 문제는 일시적 현상이 아닌 구조적 위험 요인임을 확인
2. **경제활동의 강력한 보호 효과**: 통제 변수 적용 후에도 경제활동 참여가 우울감 위험을 19.07% 감소시키는 순수 보호 요인임을 입증
3. **수면 부족의 독립적 위험 효과**: 수면 1시간 감소마다 우울감 위험이 20.08% 증가하여 수면 개선이 청년 정신건강에 직접적 영향을 미침을 확인
