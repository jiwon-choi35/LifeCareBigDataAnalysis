# 청년층 우울 위기 진단

질병관리청 **지역사회건강조사** 데이터를 활용하여 청년층(20~39세)의 우울증 발생에 영향을 미치는 요인을 로지스틱 회귀 분석으로 규명한 프로젝트입니다.

---

## 분석 배경

우울증은 최근 5년간 국내 정신질환 중 환자 수 1위(평균 980,888명)를 기록하며, 2020년 832,483명에서 2024년 1,106,603명으로 지속 증가하는 추세입니다. 특히 경제적 불안정과 수면 문제가 심각한 청년층을 대상으로 우울증 위험 요인을 데이터 기반으로 분석하고자 했습니다.

---

## 연구 질문

2. 청년층의 **수면시간**이 우울증 발생에 관여하는가? -> Code/sleep_hours_code

## 데이터

**출처**: 질병관리청 지역사회건강조사 ([https://chs.kdca.go.kr/chs/main.do](https://chs.kdca.go.kr/chs/main.do))


**수집 기간**: 2020~2024년 (5개년)


**대상**: 20세~39세 청년층

**다운로드 방법**:
1. 위 사이트 접속 후 이용계획서 제출 및 승인
2. 자료년도별(2020~2024) SAS 파일 각각 다운로드 (총 5회)
3. 접수번호 발급 후 다운로드 가능 (1인당 3회 제한)

**데이터 사용 방법**:
SAS 파일들을 다운로드 받은 후, 경로를 지정 후 사용한다.

**주요 변수**:

| 변수명 | 변수 라벨 | 처리 방식 |
|--------|-----------|-----------|
| `mt_b0100` | 우울감 경험률 (종속변수) | 1(경험) / 0(미경험) |
| `soa_01z1` | 경제활동 여부 (독립변수) | 1(예) / 0(아니오) |
| `mtc_10z1` | 수면시간 (독립변수) | 연속형 (시간) |

---

## 분석 방법

이분형 종속변수(우울증 경험 여부)에 대해 **로지스틱 회귀 분석**을 적용했습니다.
오즈비(Odds Ratio)를 산출하여 독립변수와 우울증 간의 상관성을 정량적으로 측정했습니다.

```python
# 변수 전처리
df_analysis['depressed']   = df_analysis[VAR_DEPRESSION].replace({2: 0}).astype(int)
df_analysis['econ_active'] = df_analysis[VAR_JOB_STATUS].replace({2: 0, 1: 1}).astype(int)
df_analysis['sleep_hours'] = df_analysis[VAR_SLEEP].astype(float)
```

---

## 분석 결과

### 경제활동 여부 → 우울증

| 지표 | 값 |
|------|----|
| Odds Ratio | 0.748 |
| 95% 신뢰구간 | [0.723, 0.775] |
| P-value | 0.000 |

경제활동에 참여하는 청년은 비참여 청년 대비 우울감을 경험할 오즈가 **약 25.17% 감소**합니다. P-value가 0.05보다 훨씬 작으므로 통계적으로 유의미합니다.

### 수면시간 → 우울증

| 지표 | 값 |
|------|----|
| Odds Ratio | 0.798 |
| 95% 신뢰구간 | [0.774, 0.823] |
| P-value | 0.000 |

수면시간이 1시간 늘어날수록 우울감을 경험할 오즈가 **약 20.2% 감소**합니다. P-value가 0.05보다 훨씬 작으므로 통계적으로 유의미합니다.

---

## 결론

- 경제활동을 하지 않는 집단이 활동을 하는 집단보다 우울증 발생 확률이 높다.
- 수면시간이 적을수록 우울증 발생 확률이 높다.
